# Code of Conduct

This project follows a research-collaboration standard: be precise, charitable and transparent about uncertainty.

Expected behavior:

- review mathematical claims on their merits;
- distinguish verified Lean code, pending interfaces and informal sketches;
- cite sources for nontrivial mathematical statements;
- help new contributors find small, reviewable tasks.

Unacceptable behavior:

- presenting an interface theorem as a completed proof;
- dismissive or hostile review comments;
- adding hidden assumptions or axioms to make CI pass;
- using AI-generated content without verification.
